# Exalt Logistics API

This project holds all the APIs for the logistics application but directly depends on the account service for the user authentication.

## Installation

### Database
```js
MongoDB
Redis
```
##### dev

```node
npm run dev
```

##### prod

```node
npm install -g pm2
pm2 start ecosystem.config.js
```
